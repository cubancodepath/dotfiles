This is a curated list of the apps that I use in my day to day as a software developer.

# Intro

First before everything I recommend install [Homebrew](https://brew.sh). It's is package manager for MacOS systems wihich will allow you to install many of the software appearing bellow

# Work Environments

- [Go](https://go.dev/dl/)
- [FNM](https://github.com/Schniz/fnm) (node versions)

- [Spaceship](https://spaceship-prompt.sh/getting-started/#__tabbed_1_2) (Terminal Theme) `brew install spaceship
`
